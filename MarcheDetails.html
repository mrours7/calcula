Projet Marchés BTP

Fullstack
Octobre 2025

Plateforme de gestion des Appels d’Offres & Contrats

marchebtp

Backend : Node.js – Express – PostgreSQL – JWT – REST API
Frontend : React – TailwindCSS – Hooks – Context API

DevOps / Architecture :

- Organisation modulaire (controllers, routes, middleware)
- Gestion d’erreurs centralisée
- Sécurisation de l’API (Helmet, CORS, validations)
- Migrations PostgreSQL + optimisation des requêtes SQL

Participation au développement d’une plateforme complète de gestion des marchés BTP intégrant appels d’offres, candidatures freelances/entreprises, messagerie en temps réel, notifications, contrats et signatures électroniques.

- Harmonisation d’une architecture complexe mêlant plusieurs types d’utilisateurs (freelances, entreprises, AO individu/société, admin).
- Uniformisation des IDs entre users, freelances, entreprises dans la messagerie et les notifications.
- Implémentation de workflows avancés (postulation, validation, acceptation/rejet, signature).

- Mise en place de contrôleurs robustes et transactions SQL (PostgreSQL + Node.js) pour garantir cohérence et atomicité.
- Création d’un système d’abstraction des IDs (résolution dynamique de participant) pour la messagerie et les notifications.
- Développement de modules critiques : gestion des appels d’offres, candidatures, notifications, signatures électroniques, statistiques et API sécurisée via JWT.



Ce que j’ai réalisé dans ce projet :

- Conception API + développement complet du backend
- Construction du schéma SQL (relations, contraintes, triggers)
- Implémentation des modules critiques : messaging, signatures, notifications, parrainage
- Développement des interfaces clés côté frontend (dashboards, formulaires avancés, messagerie)
- Tests, debug et amélioration des performances
Rédaction de documentation technique (endpoints, schéma DB, règles métier)
